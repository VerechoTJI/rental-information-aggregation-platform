# Decision Records

This folder tracks implementation decisions that support RIAP work without changing the SRS itself.

## Current Files

- `dbms-persistence-approach.md`: records the decision to use `JPA` with `Hibernate` for the DBMS layer.
- `decision-tracking-template.md`: reusable template for future decision records.

## Usage

1. Create a short decision record under `docs/decisions/`.
2. Keep the record aligned with the active PBI and relevant SRS requirement IDs.
3. Treat the decision as implementation guidance, not a new requirement.
